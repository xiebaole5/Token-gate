## 项目定位

TokenGate 是一个完全本机运行的 AI token 消耗管家。把任何 AI 工具（Cursor / Claude Code / Cline 等）的 base_url 指向本地代理，就能实时看到每一分钱花在哪个模型、哪个项目、还剩多少额度。预算见底自动拦截，必须人工点确认才能继续。完全本地、不联网、不上传任何数据。

## 设计规范（必须严格遵守）

### 反 AI-slop 铁律
- ❌ 禁止紫色 / AI 蓝渐变
- ❌ 禁止 Inter 字体
- ❌ 禁止 emoji（全部替换为线性 SVG 图标）
- ❌ 禁止毛玻璃效果大面积使用
- ❌ 禁止通用卡片堆叠布局

### 色彩体系
- 背景：zinc 系暗色 `#0c0f14` → `#111827` → `#1f2937` → `#374151` 四级灰
- 强调色：**Emerald 单色** `#10b981`（唯一强调色，不混用其他彩色）
- 危险：`#ef4444`（仅用于预算闸拦截/超额提示）
- 警告：`#f59e0b`（仅用于预算接近上限）
- 信息：`#06b6d4`（仅用于图表中的对比色，不用于 UI 元素）

### 排版
- 字体：`-apple-system, BlinkMacSystemFont, "SF Pro Text", "PingFang SC", "Helvetica Neue", Arial, sans-serif`
- 数据/数字：`font-variant-numeric: tabular-nums`（所有金额、token 数量必须等宽对齐）
- 中文行高 1.6 ~ 1.7，英文 1.5
- 标题字重 600-700，正文 400，辅助文字 300-400

### 图标
- 全部使用线性 SVG 图标（stroke-based，stroke-width: 1.5-2）
- 图标统一在 `src/components/icons.tsx` 里管理
- 每个 SVG 尺寸 16/20/24 三档，颜色用 `currentColor` 继承

### 间距与分组
- 用分隔线（1px solid var(--line)）和负空间分组，不用大圆角卡片套卡片
- 模块之间用 1px 分隔线，不用 box-shadow 层叠
- 信息密度高，但留白节奏清晰

## 当前页面结构

项目有 4 个 tab：
1. **总览（Dashboard）** — 驾驶舱首页
2. **接入与监听（Providers）** — API provider 管理 + 代理地址
3. **流水与闸门（Records）** — 消耗流水 + 闸门拦截日志
4. **AI 管家（Butler）** — 本地 LLM 问答 + AI 工具扫描

## 需要你做的 5 件事（按优先级排序）

### 1. 驾驶舱首页重构（最高优先级）

当前首页信息太挤、太散。改造成金融驾驶舱风格：

**布局：**
```
┌─────────────────────────────────────────────────┐
│ 指标带（6 项，用分隔线分组，不用卡片）              │
│ 本月消耗 | 剩余预算 | 已拦截 | 活跃工具 | 今日消耗 | 7日趋势 │
├───────────────────────┬─────────────────────────┤
│                       │                         │
│  14 天消耗趋势大图     │  额度健康度仪表盘        │
│  （ECharts 面积图）    │  （圆环 + 百分比）       │
│                       │                         │
├───────────────────────┴─────────────────────────┤
│ 模型 Token 排行榜（Top 5，紧凑横条）              │
│ + 输入 vs 输出 token 对比图（ECharts 柱状图）     │
└─────────────────────────────────────────────────┘
```

**指标带样式：** 每个指标就是 `数字（大、粗、tabular-nums） + 标签（小、灰）`，指标间用 1px 竖线分隔，不要边框不要背景不要圆角。

**ECharts 配色：** 全部用 Emerald 渐变（`#059669` → `#10b981` → `#34d399`），不用彩虹色。警戒线用 `#ef4444` 虚线。

### 2. SVG 图标模块

当前代码里部分地方还在用 emoji 或文字。请：
- 在 `src/components/icons.tsx` 里补齐所有需要的图标（盾牌、锁、图表、闪电、扫描、机器人、闸门、美元、人民币、复制、刷新、设置、加号、箭头等）
- 全站扫描，把所有 emoji 替换成 `<IconXxx />` 组件
- 图标统一 20x20，颜色用 `currentColor`

### 3. 预算闸拦截弹窗视觉升级

当前 `GateModal.tsx` 是基础弹窗。升级为：
- 弹窗背景加一层红色径向渐变光晕（subtle，不是全红）
- 金额数字更大更醒目，超额数字用 `#ef4444` + 加粗
- 「取消阻断」按钮用 zinc 灰，「确认放行」按钮用 `#ef4444` 实色（不是绿色——这里放行是危险操作）
- 弹窗边框用 `1px solid #ef4444`
- 底部加一行小字：「此操作将记录在闸门日志中」

### 4. Provider 卡片和流水列表美化

**Provider 卡片：**
- 每张卡片左上角一个状态点：绿色 = 代理在线，灰色 = 未连接
- 代理地址用等宽字体展示，旁边一个复制按钮（点击后变 ✓ 1.5 秒再变回）
- 卡片间用 1px 分隔线，不要大间距

**流水列表：**
- 表头固定，内容区滚动
- 金额列右对齐，tabular-nums
- 闸门拦截记录行左边加一个红色竖条标识
- 时间列用相对时间（「3 分钟前」「2 小时前」），hover 显示绝对时间

### 5. AI 管家面板

- 顶部一个「隐私徽章」：盾牌图标 + 「数据不出本机」文字，Emerald 色小标签
- 大脑切换（本地 LLM / 云端）用 segmented control 样式，选中态用 Emerald 填充
- 本地模型状态：绿灯 = 在线，黄灯 = 加载中，灰灯 = 未安装
- 对话区用气泡样式，AI 回复左对齐，用户输入右对齐
- 底部输入框和快捷问题按钮组
