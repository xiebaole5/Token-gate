### 【标题】
学习工作 · TokenGate — 给 AI 花钱装个闸,先问过你才能继续

### 【正文】

#### 1. 创意名称 + 创意介绍

**TokenGate** — 一个完全本机运行的 AI token 消耗管家。

**想解决什么问题:** 我自己电脑上挂着 11 个 AI 工具(Cursor / Claude Code / TRAE / Cline / Codex CLI / OpenClaw / MLX 模型缓存…),每天用 DeepSeek / 通义 / Kimi / GPT / Claude 等多家 API。**我最害怕的不是花了多少钱,而是不知道花了多少**——每家账单分散在不同后台,按量付费时一次 Cline 跑 agent 任务能烧掉 ¥30 我完全无感,等月底信用卡账单来才心惊肉跳。

**为什么会想到做这个:** 更深一层是 AI agent 越来越自主——你给它一句话,它在后台 reasoning 几十轮、调几百次工具,**目前市面上没有任何机制让你在它失控时叫停**。这跟一辆没有刹车的车没区别。

**大概是什么产品:** 浏览器打开即用的本地 Web 工具(不上传任何数据,只绑 127.0.0.1),核心是一个"代理 + 闸门",把 AI 工具的 base_url 指过来就能实时计量 token 消耗,预算见底自动弹窗拦截,**必须人工点确认才能继续花钱**。

#### 2. 目标用户及痛点

**面向哪些用户:**
- 同时挂多个 AI 工具(Cursor / Cline / Aider / Claude Code 等)、月底信用卡总让人心惊的开发者
- 用 AI agent 替自己干活、担心它失控烧爆预算的创作者
- 介意把 API key 上传给第三方 SaaS 的隐私敏感用户

**在什么场景下使用:**
- 写代码时挂着 Cursor + Cline,想知道哪个工具最烧钱、哪个项目占用最多
- 用 AI agent 跑长任务,想给它设个"花到 ¥50 就必须停下来问我"的硬上限
- 月末复盘:这个月我在 8 家大模型上一共花了多少、哪家性价比最高

**当前痛点:**
1. **账单分散** — 国内 DeepSeek / 通义 / Kimi / 智谱 / 豆包 / MiniMax 每家都要单独登后台查余额,**根本没人有耐心一家家点**
2. **无感扣费** — 按量付费 + AI 自主调用,等你发现已经晚了
3. **多项目混账** — 同一个 key 被多项目共用,完全不知道哪个项目最烧钱
4. **AI 失控没刹车** — 现有工具(Helicone / Langfuse / Portkey)都是事后记账,**没有一款主动拦截**
5. **隐私焦虑** — 不敢把所有 key 上传给第三方 SaaS

#### 3. 价值与意义

**效率价值:**
- **一个面板看清所有 AI 消耗** — 不再需要登 8 家网站查余额
- **代理监听零侵入** — 任何 OpenAI 协议兼容的工具(国内外几乎全部主流)改一个 base_url 就能接入,实时捕获真实 token 用量
- **一键扫描 65 种本机 AI 工具** — 自动发现你装了什么,降低接入摩擦

**社会/行业价值:**
- **第一个把"AI 失控"焦虑变成具体工程护栏的工具** — "预算见底 → 自动拦截 → 必须人工点放行才能继续"——这条护栏是市面上 SaaS 类竞品(Helicone / Langfuse / Portkey)从未做过的
- **填补国内开发者的空白市场** — 现有同类产品全是海外 SaaS,**没有一款纯本地、不联网、专为国内国外双币种设计的 token 管家**
- **隐私承诺写进代码** — 后端只绑 127.0.0.1、文件扫描只看目录名不读内容、AI 管家上下文是只读账本快照——**让"AI 工具可以不需要 SaaS"成为可能**

**核心哲学(差异化):**
**"AI 是执行者,人是立法者。"** 你设上限,闸门拦截,放行必须经过你。这不只是一句口号,是写在代码里的工程护栏。

## B 部分 · 喂给 TRAE Work 的 prompt

打开 https://www.trae.cn/TRAE → 选 **Auto 模式** → 粘贴下面这段:

```
请帮我生成一份「TokenGate」项目的创意提案 HTML 单文件,用于参加 TRAE AI 创造力大赛报名。要求:

【项目定位】
TokenGate 是一个完全本机运行的 AI token 消耗管家。把任何 AI 工具(Cursor / Claude Code / Cline / Aider 等)的 base_url 指向 TokenGate 代理,就能实时看到每一分钱花在哪个模型、哪个项目、还剩多少额度,预算见底自动拦截,必须人工点确认才能继续。完全本地、不联网、不上传任何数据。

【核心哲学】
"AI 是执行者,人是立法者。" 你设上限,闸门拦截,放行必须经过你。这是市面上唯一一个把"AI 失控"焦虑变成具体工程护栏的工具。

【目标用户】
- 同时挂多个 AI 工具、月底信用卡总让人心惊的开发者
- 用 AI agent 替自己干活、担心它失控烧爆预算的创作者
- 介意把 API key 上传给第三方 SaaS 的隐私敏感用户

【5 大功能】
1. 驾驶舱总览:6 项指标带 + 14 天大趋势 + 每个 API 自动一张迷你计量卡 + 模型 Token 排行榜 + 输入vs输出对比图
2. 代理监听:OpenAI 协议兼容,任何工具改一个 base_url 就能接入,实时捕获真实 token 用量
3. 预算闸 + 人工放行:预算见底自动拦截,必须你点确认才能继续花钱,每次拦截留痕
4. AI 工具自动扫描:一键扫描 Mac 上 65 种已知 AI 工具,自动建 provider 骨架
5. 本地 AI 管家:用本地大模型(Ollama / MLX / LM Studio)问账本,数据不出门;云端模型必须确认才外发

【双币种设计】
国内模型 ¥/M、国外模型 $/M 两张单价表,自动按汇率(1 USD ≈ 7.2 CNY)换算,统一入账。

【数据洞察案例】
同样 146 个输出 token,DeepSeek $0.000166 vs GPT-4o $0.00149,DeepSeek 便宜 9 倍——这就是 TokenGate 的核心价值证明。

【视觉风格】
- 暗色驾驶舱主题(zinc 系背景 #0c0f14)
- Emerald 单色强调(#10b981)
- 零 emoji 全 SVG 图标
- 高密度信息排版,反 AI-slop 风格
- 数据图表为主,文字辅助

【HTML 要求】
- 单文件 self-contained(可内联 CSS,可用 CDN 引入 ECharts 或纯 SVG 模拟图表)
- 至少 5 屏内容:Hero / 痛点 / 解决方案 / 核心功能 / 隐私承诺
- 不少于 3 个数据可视化模块(可以是模拟数据)
- 适配桌面端展示
- 整体调性专业、克制、像金融驾驶舱

请直接给我能下载的 HTML 文件。
```

## 你下一步的完整流程(20 分钟搞定)

1. **打开 https://www.trae.cn/TRAE,选 Auto 模式**
2. **粘贴 B 部分的 prompt**,等几分钟它生成 HTML
3. **下载生成的 HTML 文件**(本地存一份,文件名建议 `TokenGate-报名创意.html`)
4. **打开 https://forum.trae.cn/c/38-category/39-category/39**(大赛报名专区)
5. **点"发新帖"**,标签勾 `学习工作`,标题贴 A 部分给的
6. **正文整段复制 A 部分的「1.~4.」**
7. **HTML 文件**:点编辑器附件上传按钮,把刚才下载的 HTML 文件传进去
8. **发布等审核**(官方每个工作日审核,一般次日有结果)
9. **审核通过后**(短信通知 + 站内消息) → 立刻去「初赛 Demo 专区」用我们已经准备好的 [TRAE-SUBMISSION.md](./TRAE-SUBMISSION.md) 发 Demo 帖
