# TokenGate · 参赛提交文档

> 给 AI 花钱装个闸 — 一个本机运行的 token 消耗管家

---

## 一、项目一句话

**TokenGate 让 AI 花你的钱之前,先问过你。**

它是一个完全运行在本机的 token 消耗管家:把任何 AI 工具接入它的代理,就能实时看到每一分钱花在哪个模型、哪个项目;预算见底时自动拦截,必须人工点放行才能继续。**不联网、不上传、不被训练**。

---

## 二、解决什么问题

**AI 时代的"无感扣费"已经失控。**

普通开发者今天平均挂着 5-10 个 AI 工具(Cursor / Claude Code / Cline / Aider / 各家 GUI / 本地推理…),每一个都在悄悄烧 token。具体痛点:

1. **账单分散**:DeepSeek 一个后台、通义一个后台、Moonshot 一个后台、OpenAI 一个后台,没人有时间挨个登录查
2. **按量付费没感觉**:一次 Cline 跑一个 agent 任务可能烧掉 ¥30,你完全无感,直到月底信用卡账单来
3. **多项目混账**:同一个 key 被多个项目共用,根本不知道哪个项目最烧钱
4. **AI 自主行动可能失控**:你给 agent 一句话,它后台 reasoning 几十轮、调几百次工具,**没有任何机制让你在它失控时叫停**
5. **隐私焦虑**:市面上有云端账单聚合工具,但要把所有 API key 上传给第三方,等于把命根子交出去

TokenGate 的设计哲学是 **"AI 是执行者、人是立法者"**:你设上限,闸门拦截,放行必须经过你。这条理念叫不动的话,AI 用得越深的人,就越被自己的工具反噬。

## 三、产品演示路径(30 秒跑通闭环)

```bash
git clone <仓库地址>
cd tokengate && npm install && npm start
# 浏览器打开 http://localhost:5173/
```

**演示 1 · 工具扫描(0 配置)**
打开首页 → 顶部「检测到的 AI 工具」自动扫到 11 个你已装的 AI 工具(Cursor / Claude Code / TRAE / Codex CLI / MiniMax 海螺 / OpenClaw / MLX 等),每个都有「一键建骨架」按钮。

**演示 2 · 代理监听 + 实时计量(无需联网)**
1. 「接入与监听」加一个 mock provider:`http://127.0.0.1:8787/api/_mock`,套餐 ¥100、额度 5、模型填 `gpt-4o, deepseek-chat`
2. 复制代理地址,在终端 curl 3 次不同模型
3. 回首页看图表跃动:6 项指标实时跳数、14 天大图实时画线、provider 迷你卡花费数字往上涨、模型 token 排行榜重排
4. 一个模型对比直击灵魂:**同样 146 个输出 token,DeepSeek $0.000166 vs GPT-4o $0.00149,便宜 9 倍**

**演示 3 · 灵魂功能(预算闸 + 人工放行)**
1. 「预算与闸门」给整体设上限 $0.005(故意设得极低)
2. 「记一笔」录一笔 GPT-4o 200 input / 1000 output(花费会超过 $0.005)
3. **弹窗拦截**:「已用 $0.0030 + 本次 $0.0150 > 上限 $0.005,需要继续吗?」
4. 你点「批准超额放行」才写库,且**留痕到"闸门记录"**,谁拦了、谁放了、什么时候、为什么

**演示 4 · AI 管家(本地大模型 + 隐私承诺)**
1. 「AI 管家」自动探测本机 Qwen3.6 / Ollama / LM Studio / MLX 端口,绿灯显示「🟢 本地模型 · 数据不出门」
2. 点快捷问题「哪个项目最烧钱」,本地大脑只拿账本快照分析,**不读任何 API key/对话历史/文件内容**
3. 如果切到云端模型,**发送前必须先弹窗确认**:「这次会把账本快照外发到 ☁️XXX」

## 五、创新点 / 差异化

### 1. 灵魂哲学:AI 是执行者,人是立法者
市面上所有 AI 消耗工具都是**事后记账**——你已经花完了它告诉你花了多少。TokenGate 是**事前拦截**——预算见底时主动拦下,**必须人工点放行**才能继续。这是把"AI 失控"这个公共焦虑,变成一条具体可执行的护栏。

### 2. 本地优先 + 零数据上传
对标产品(Helicone、Langfuse、Portkey、OpenRouter)都是 SaaS,必须把 API key 和调用日志上传到他们的服务器。TokenGate **从代码到数据全在你电脑里**,后端只绑 `127.0.0.1`。这是国内开发者特别需要、但目前完全空白的形态。

### 3. 中国市场专属设计
- 国内/国外两张单价表,人民币按汇率自动换算
- 内置 11 种国产 AI GUI 客户端扫描(Kimi/豆包/通义/文心/元宝/智谱/DeepSeek/海螺/讯飞/小米 MiMo/OpenClaw)
- 全 UI 中文,无翻译腔
- 适配国内 base_url(百炼 / 火山引擎 / 智谱 / 月之暗面 / DeepSeek 等)

### 4. 设计语言反 AI-slop
基于 taste-skill 开源规范实施:**零 emoji 全 SVG 图标 + Emerald 单色强调 + 高密度驾驶舱 + 去圆角卡片堆叠**。看起来不像"另一个 AI 默认 UI",像专业财务工具。

### 5. AI 工具自动发现
65 种 AI 工具的扫描清单,覆盖国内外几乎所有主流形态(IDE / 编辑器扩展 / CLI / GUI / 本地推理 / 模型缓存 / Agent 框架)。扫到就一键建骨架,**用户只需要填 API key**,把"接入摩擦"压到最低。

## 六、技术架构

```
┌─────────────────────────────────────────────────┐
│  浏览器 · http://localhost:5173                  │
│  React 19 + Vite + ECharts + TypeScript         │
└──────────────────┬──────────────────────────────┘
                   │ HTTP + SSE 实时推送
┌──────────────────▼──────────────────────────────┐
│  本机后端 · http://127.0.0.1:8787 (仅本机)        │
│  Express 5 + tsx watch                          │
│                                                  │
│  ┌─────────────────────────────────────────┐    │
│  │  /proxy/:providerId/v1/*                │    │
│  │  → 透传到上游真实 API                     │    │
│  │  → 解析 usage / SSE 流                   │    │
│  │  → 实时计费、扣减额度、拦截超额            │    │
│  └─────────────────────────────────────────┘    │
│                                                  │
│  ┌─────────────────────────────────────────┐    │
│  │  /api/scan/ai-tools                     │    │
│  │  → 只读目录是否存在,绝不读文件内容          │    │
│  └─────────────────────────────────────────┘    │
│                                                  │
│  ┌─────────────────────────────────────────┐    │
│  │  /api/butler/*                          │    │
│  │  → 本地模型探测 / 账本快照转发              │    │
│  └─────────────────────────────────────────┘    │
└──────────────────┬──────────────────────────────┘
                   │ 持久化
┌──────────────────▼──────────────────────────────┐
│  ~/.tokengate/state.json (本机文件)              │
└─────────────────────────────────────────────────┘
```

**关键工程取舍**:
- **不用数据库**:一个 JSON 文件搞定,降低部署复杂度,符合"自用 alpha"定位
- **不开外网端口**:仅绑 `127.0.0.1`,扫描端口也只对本机响应
- **SSE 推送**:前端不轮询,代理捕获到一笔 usage 立即推送给所有打开的页签
- **OpenAI 协议兼容**:不发明新协议,直接复用国内国外所有兼容上游

## 七、隐私 / 安全承诺

| 承诺 | 工程实现 |
|---|---|
| 不联网、不上传 | 后端只绑 `127.0.0.1`;前端不发任何第三方请求 |
| 不读 API key | key 存 `~/.tokengate/state.json`,UI 只展示 mask 后字符 |
| 不读对话历史 | 代理转发用流式管道,从不缓存 body |
| 不读文件内容 | 工具扫描用 `fs.existsSync` 和 `fs.readdirSync` 目录名匹配,**永不读文件** |
| 不被训练 | 本地数据 = 本地数据,不存在被任何模型采集的通道 |
| 一键清空 | `rm ~/.tokengate/state.json` 全部归零 |

## 附:作品对应解决的三个时代命题

1. **AI 自主行动失控**:用"闸门 + 人工放行"做出第一条具体可执行的护栏
2. **数据隐私焦虑**:用"本地运行 + 零上传"证明 AI 工具可以不需要 SaaS
3. **创作者经济成本黑箱**:用"驾驶舱可视化 + 实时计量"让 token 这个隐形消耗变成看得见的财务

> AI 工具会越来越多、越来越自主、越来越烧钱。  
> TokenGate 不阻止你用 AI,只是替你守住"你说继续才继续"这条线。
